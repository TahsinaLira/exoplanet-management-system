

ExoDiscover

A simple web-based project to explore exoplanets and related research papers. Built using Java.


Tech Used:

- Java
- JavaScript
- CSS
- MySQL
- Apache Tomcat

How to Run--

 Clone the repo

 Import into Eclipse/IntelliJ
 
 Set up MySQL database

 Run on Tomcat server
